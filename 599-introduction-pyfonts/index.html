Introduction to PyFonts

PyFonts is a Python library for loading fonts in matplotlib charts. It provides a simple way to access any fonts you can imagine.

This post will teach you how to use this library and all its main features, with concrete example and reproducible code.

Libraries

First, you need to install pyfonts with the following command: pip install pyfonts.

With pyfonts you can easily change the font in your matplotlib charts. It was created by Joseph Barbier.

pyfonts provides 2 functions: fig_arrow() (draw an arrow on a matplotlib figure) and ax_arrow() (draw an arrow on a matplotlib axes). We'll also need to load matplotlib to create the figure.

import matplotlib.pyplot as plt
from pyfonts import load_font

Quick start

The easiest way to get started is to use a font available on Google font repo and add ?raw=true at the end of the url.

# load font
font = load_font(
   "https://github.com/google/fonts/blob/main/apache/ultra/Ultra-Regular.ttf?raw=true"
)

# check how the font looks on a minimalist example
fig, ax = plt.subplots(figsize=(10, 6))
ax.text(
   x=0.5,
   y=0.5,
   s=f"What an easy way to load fonts, isn't it?",
   font=font,
   fontsize=20,
   ha="center",
)
plt.show()

How to find fonts

Browsing fonts

There are many fonts available on the web. The easiest way to find one is to follow these steps:

  • Browse Google Font website to find a font that you like. Let's say you want to use Amaranth.
  • Go to Google font github repository and type the name of your desired font in the search bar. We find that Urbanist font file in Bold is named https://github.com/google/fonts/blob/main/ofl/amaranth/Amaranth-Bold.ttf.
  • copy the url and add ?raw=true at the end, which gives us https://github.com/google/fonts/blob/main/ofl/amaranth/Amaranth-Bold.ttf?raw=true
  • and that's it! Just pass this to load_font() to use it in your matplotlib charts

You can find other fonts at:

  • Awesome fonts
  • Nerd fonts

Specifying fonts for PyFonts

For the url to be readable by PyFonts when using a Github url, it must be in one of (the following 3 urls are completely equivalent):

  • https://github.com/google/fonts/blob/main/apache/ultra/Ultra-Regular.ttf?raw=true
  • https://github.com/google/fonts/raw/main/apache/ultra/Ultra-Regular.ttf
  • https://raw.githubusercontent.com/google/fonts/raw/main/apache/ultra/Ultra-Regular.ttf

The recommended is the first (https://github.com/google/fonts/blob/main/apache/ultra/Ultra-Regular.ttf?raw=true) because you just need to add ?raw=true after the end of the Github url.

Different weight and style

When you load a font, you don't load all its extensions: bold, italic, thin etc, but only the one from the url. If you want to be able to use a font and its bold version, for example, you need to load both fonts:

font = load_font(
   "https://github.com/google/fonts/blob/main/ofl/amaranth/Amaranth-Regular.ttf?raw=true"
)
bold_font = load_font(
   "https://github.com/google/fonts/blob/main/ofl/amaranth/Amaranth-Bold.ttf?raw=true"
)

fig, ax = plt.subplots(figsize=(10, 6))
ax.text(
   x=0.5,
   y=0.5,
   s=f"Congrats, you now have a cool font!",
   font=font,
   fontsize=20,
   ha="center",
)
ax.text(x=0.5, y=0.3, s=f"And now it's bold", font=bold_font, fontsize=20, ha="center")
plt.show()
